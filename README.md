# Retail Sales Data Analysis Dashboard

## Project Overview

This project focuses on Retail Sales Data Analysis using Excel and Dashboard Visualization.

The dataset contains customer transactions including:

* Product Category
* Gender
* Age
* Quantity
* Price per Unit
* Total Amount

## Key Analysis Performed

* Sales analysis by Product Category
* Gender-wise sales comparison
* Monthly sales trend analysis
* Revenue and quantity insights
* Category-wise performance breakdown
* Interactive Dashboard creation

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Dashboard Visualization
* Data Cleaning

## Project Highlights

* Identified top-performing product categories
* Compared male vs female purchasing behavior
* Analyzed monthly sales performance
* Created an interactive dashboard for business decision-making
